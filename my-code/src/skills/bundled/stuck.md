We seem to be stuck. Let's take a step back and reset our approach. 
1. Summarize the root goal we are trying to achieve.
2. List all the assumptions we've made so far.
3. Identify which assumption might be incorrect.
4. Propose a completely different approach or test we can run to narrow down the issue.
